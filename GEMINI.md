# Gemini Instructions

When working in this repository, please follow these guidelines:

1. **Modern Go**: Write code using modern Go patterns and features.
2. **Test-Driven Development (TDD)**: Always add a failing test first before writing any functional code.
3. **Workflow**: Once any change is complete and all tests pass, follow the [finish.md](file:///workspaces/recordalerting/.agents/workflow/finish.md/finish.md) workflow to commit and push your changes.
